# Road Risk Estimation System

A machine learning based web application that estimates accident risk for a selected road route.

## Features
- Route segmentation
- Real-time weather integration (Open-Meteo)
- Road classification using OpenStreetMap
- Machine learning risk prediction
- User login and search history

## Tech Stack
Backend: Flask
Frontend: Leaflet.js
Machine Learning: Random Forest
APIs: Open-Meteo, Overpass API

## How to Run
pip install -r requirements.txt
python backend/app.py
